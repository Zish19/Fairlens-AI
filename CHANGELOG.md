# Changelog

## [1.0.0] - 2026-07-20

### Added
- Fairness analysis using Fairlearn
- SHAP explainability for local and global feature importance
- AI-generated fairness insights with graceful fallback
- FastAPI backend with modular layered architecture
- React frontend dashboard
- Background worker infrastructure using Celery and Redis
- JWT Authentication and user management
- Docker Compose deployment orchestration
- Prometheus monitoring and structured logging (structlog)
- CI/CD with GitHub Actions including Alembic and Docker smoke tests
